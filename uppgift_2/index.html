First School Project:
Responsive Web Design with HTML & CSS For my first school project, I built a fully responsive website using semantically written HTML and CSS. The project emphasized the correct use of HTML5 tags to structure content clearly, ensuring that the site was accessible and easy to navigate. Additionally, the website was designed to be adaptable to any screen size, maintaining functionality and appearance down to a 400px width for mobile devices. Through this project, I gained a solid foundation in responsive design principles and the importance of using semantic code for improved SEO and accessibility.
